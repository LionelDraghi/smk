smk command line

smk command line
----------------

```
smk -h
```

```

Usage : smk [Options] make_file

Options :
   -We | --Warnings=error : treat warnings as errors
   -v  | --verbose
   -q  | --quiet          : no message unless error.
                            Warning are also ignored.
         --version        : smk version
   -h  | --help           : this message

https://github.com/LionelDraghi/smk

```
